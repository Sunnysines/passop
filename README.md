This is a password Manager.

Pass Saver- Your Own Password Manager. The following are the features that you get:-

👉 All your passwords in one place.

👉 You can manage, edit and delete passwords.

👉 You can get to download a pdf of your passwords.

👉 It uses local storage, which remains with you until you delete your browser cache. Or manually delete it from local storage.

👉 A hyperlink to a website to check whether your password has been hacked and used elsewhere.

👉 Responsive website, will work on mobile devices.
